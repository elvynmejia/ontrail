# project setup
Clone this repo into your vm working directory  

`git clone https://github.com/elvynmejia/ontrail.git`  
`vagrant ssh # ssh into your vm`  
`cd /vagrant/ontrail`  

# intall dependecies  
`make install`  

# run app
`make run`

# seed database
`make seed`

# clean
`make clean`

# run tests
py.test
